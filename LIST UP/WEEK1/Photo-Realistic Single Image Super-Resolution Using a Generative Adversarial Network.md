# GAN을 이용한 단일 이미지 초고해상도 복원 <br>
## Abstract <br>
현재 SR이 해결해야 할 문제는 대규모의 업스케일링에서 미세한 질감 디테일을 어떻게 복원할 것인가이다. 최근의 연구들은 MSE를 최소화하는데에 초점을 두고 있다. 결과 추정치는 피크 신호 대 잡음 비율이 높지만 고주파 세부 정보가 부족한 경우가 많으며 고해상도에서 기대하는 정확도에 미치지 못한다.
